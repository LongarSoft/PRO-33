Nuevo portal web para firma de abogados

La presente propuesta de desarrollo de página web surge como respuesta a la necesidad de nuestra cliente de establecer una presencia en línea efectiva para promover sus servicios profesionales en diversas áreas, incluyendo el ámbito legal y otros campos como ingeniería civil, contabilidad, medicina forense, psicología forense, entre otros.

Objetivos:

  • Diseñar una interfaz intuitiva y atractiva que refleje la imagen profesional de la firma.
  • Integrar secciones específicas para cada área de servicio, incluyendo información detallada y contactos.
  • Implementar un espacio de noticias para mantener a los visitantes informados sobre novedades relevantes.
  • Incluir un formulario de contacto general para consultas y solicitudes de información.
  • Incorporar imágenes que complementen la presentación de los servicios y el equipo de trabajo.
  • Agregar biblioteca de contenido como revistas y PDFs relevantes para el cliente.

1 Organización de contenidos
Se enviará una lista de requerimientos para la recopilación del contenido y se establecerá, mediante reuniones y asesorías, los pasos a seguir para cumplir con los objetivos del sitio web.

Posteriormente, se creará un esquema jerárquico de las páginas (e.g. sobre nosotros, beneficios, casos de uso), campos de formularios, lógicas, registros y navegación del sitio web mediante un sitemap.

Finalmente, se organizarán los contenidos en distintas carpetas Drive para su posterior desarrollo.

2 Diseño web
Elaboración de la línea gráfica del sitio web, teniendo en cuenta el manual de marca proporcionado por el cliente, el customer journey y las llamadas a la acción que se buscan cumplir. Se procederá con el diseño de las siguientes plantillas:
  • Inicio
  • Programación
  • En demanda / On demand
  • Página informativa (ej. Sobre nosotros)
  • Guía de estilos (tipografías, colores, botones)
El diseño será mostrado a su equipo para llegar a una experiencia web única. Visualmente, será adaptable a dispositivos en escritorio, tablet y móvil.
Ejemplos: Laboratorio Omega | Union Agronegocios 		1 2100 2100
3 Diseño web
Basándonos en la línea gráfica aprobada, se desarrollará el sitio web con el sistema de contenidos WordPress.

Contará con los tipos de contenidos, páginas y secciones definidas previamente en el sitemap. Las páginas podrán contener texto, galería de imágenes/videos, mapas y formularios. Serán comprimidas y optimizadas para una óptima velocidad de navegación.

Al finalizar el desarrollo, se establecerá un proceso de integración con herramientas web (Analytics, Tag Manager, Meta Pixel, Search Console para posicionamiento de palabras claves, etc.) junto a su equipo de comunicación para contar con reportes de las métricas de visitas e integración con redes sociales.
Ejemplos de sitio web: Grupo Venado | DMI 		1 2100 2100
4 Base de Datos
Haremos la creación e implementación de una base de datos con la cual se pueda administrar de manera segura y eficiente secciones solicitadas:
  • Equipo de trabajo (nombre completo, hoja de vida, imagen de perfil, área de especialización, información de contacto, enlaces a redes sociales profesionales, ejemplos de trabajos realizados, certificaciones y testimonios de clientes)
